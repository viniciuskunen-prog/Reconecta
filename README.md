# Reconecta Landing Page

Landing page em React + Vite para o evento Reconecta.

## Rodar localmente

```bash
npm install
npm run dev
```

## Gerar build

```bash
npm run build
```

## Assets

Os arquivos de marca ficam em:

```txt
public/assets/
```

Troque os placeholders de data, local, endereço, fotos dos facilitadores e link do botão de inscrição direto em `src/main.jsx`.
